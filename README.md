# FemgameLan Website

Offisiell webside for FemgameLan 2026 - et inkluderende LAN-arrangement for jenter og ikke-binære gamere.

## 📋 Innhold

- **index.html** - Hovedulside med all informasjon
- **style.css** - Styling med farger fra logoen (cyan, magenta, lilla)
- **script.js** - Interaktivitet og animasjoner
- **logo.png** - FemgameLan logo (last opp denne)

## 🎨 Fargeskjema

- Cyan: #00d4ff
- Magenta: #ff006e
- Lilla: #a000ff
- Hvit: #ffffff
- Mørk bakgrunn: #0a0e27

## 📱 Responsivt design

Websiden er fullt responsiv og fungerer perfekt på:
- Desktoper
- Tablets
- Mobiltelefoner

## 🚀 Instruksjoner for oppdatering

1. Last opp logofilen din som `logo.png` i rotmappen
2. Oppdater kontaktinformasjon i kontaktseksjonen
3. Legg til påmeldingsfristen i påmeldingsseksjonen
4. Legg til sponsorer når de bekreftes

## 📂 GitHub Pages

Websiden hostes automatisk på GitHub Pages. Den vil være tilgjengelig på:
`https://idaeliseenger-pixel.github.io/femgame/`

For å få den til å fungere på femgame.no, må du konfigurere DNS-innstillingene dine til å peke på GitHub Pages.
